## ðŸ”¹ Getting Started

### Prerequisites

- Node.js >= 16.x
- Angular CLI >= 15.x
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/HasiniSooorige/chat-ai-bot.git
cd chat-ai-bot
```
2. Install dependencies:

```bash
npm install
```

3. Run the Angular application:

```bash
ng serve
```

4. Open your browser at ``http://localhost:4200``
